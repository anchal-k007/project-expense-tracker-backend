## Environment variables required to run this repo

```
MONGODB_CONNECTION_URL=<Your-mongodb-connection-string>
MONGODB_PASSWORD=<Your-mongodb-password>
MONGODB_USERNAME=<Your-mongodb-username>
MONGODB_COLLECTION_NAME_DEV=<Development-database-name>
MONGODB_COLLECTION_NAME_PROD=<Production-database-name>

JWT_SECRET_STRING_DEV=<Secret-key-used-for-authentication-in-dev-landscape>
JWT_SECRET_STRING_PROD=<Secret-key-used-for-authentication-in-prod-landscape>

MONGODB_ADMIN_USERNAME=[OPTIONAL]<mongodb-admin-username>[not used anywhere]
MONGODB_ADMIN_PASSWORD=[OPTIONAL]<mongodb-admin-password>[not used anywhere]
```
